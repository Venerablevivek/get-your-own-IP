# sysytem-ip [![Build Status]]

> An NPM Package to get your own IP Address


## Install

```
$ npm install system-ip
```


## Usage

```js
import {myIP} from "get-your-ip";

const ans = myIP;
console.log(ans);
//=> ['192.168.9.104', '192.168.9.104']
```

## API

### myIp

Returns IPv4 address of your system.
